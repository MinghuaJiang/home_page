Tools

BoxLink

This is a file sharing app based on Box API. It was created to reduce the effort to share file using UVA Box. Web browser is no longer needed to be opened to share the file.
OAuth2 was used to authenticate and authorize my application to access the Box content.
File can be shared by either using drag and drop or opening a file chooser dialog.

Java

jTakeColor

This is a color taking tool. It was created to support color taking on any objects on the screen and then copy the color into CSS for web development.
Color can be represented by both HTML mode or RGB mode.
Color that was taken are recorded on the right side of the tool.
Color palette function is also implemented to enable user to get a color similar to the color that was taken by modifying each channel a little bit.
